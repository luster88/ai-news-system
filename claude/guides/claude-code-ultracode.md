## 2026-05-31

### Asked Claude Code for a "deep search" in ultracode mode — it spun up ~70 agents across a 4-phase pipeline on its own

Reddit ユーザーが Claude Code の ultracode モードで「deep search」を依頼したところ、Claude が自律的に約70個のエージェントを起動し、4段階（discovery → benchmark → enrich → verify）のパイプラインを構築した事例報告。ultracode モードはオーケストレーション計画をスクリプト化することでコンテキストウィンドウの制約を回避するが、各エージェントごとにコンテキスト設定のコストが発生するトレードオフがある。大規模タスクには有効だが、小規模な修正には通常セッションの方が効率的。
