### ClaudeCodeでviライクエディタ「FoxEditor」を作ってみた#1

Windows環境でviライクな操作体系を持つシンプルなテキストエディタ「FoxEditor」をClaude Codeを使って自作した記事。Python + tkinterで実装され、モード（Normal/Insert/Command）を枠線の色で視覚的に区別できる点が特徴。Claude Codeの丁寧なコード出力により、エディタ開発の敷居が下がったことを強調している。
