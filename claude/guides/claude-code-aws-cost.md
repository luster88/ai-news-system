### Fable5を利用してAWSのクラウド利用費を削減する

Claude Code（Fable 5モデル）を使用してAWSのコスト分析から削減候補の特定、削除スクリプト生成までを半日で実現した事例。読み取り専用権限のIAMユーザーを作成し、Cost Explorer APIでコストを多角的に取得。AIは分析とスクリプト生成のみを行い、実行は人間が担当する安全な設計。一連の手順をClaude CodeのSkillsとして固定化し、/aws-cost-reportコマンドで再現可能にしている。
