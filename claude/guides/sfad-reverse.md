### 引き継ぎ地獄エンジニアのためのSFAD ― 仕様書ゼロのコードを前にして最初にやること

仕様書のない引き継ぎコードに対し、Claude Code の /sfad reverse コマンドで既存コードから Example Map 形式の仕様書を自動生成する手法を解説。7つのフェーズで仕様を逆算し、マジックナンバーやセキュリティリスクを可視化。引き継ぎ初日の4時間でやるべきステップと、テスト生成による安全なリファクタリング基盤の作り方を具体例とともに紹介。
