## 2026-07-16

### I built a true-scale atlas of the universe (8.4M real stars) in about a week with Fable

ユーザーが Fable（Claude Code の機能）を使用して、約1週間で実スケールの宇宙アトラスを構築したプロジェクト事例。840万の実在する星、260万の銀河、惑星の実際の軌道、リアルタイム衛星などを含む WebGPU ベースのインタラクティブな可視化ツールを開発。92のプルリクエスト、237コミット、約14,500行の TypeScript/WGSL コードを生成し、物理演算やデータパイプラインもほぼ全て Fable が実装した。
