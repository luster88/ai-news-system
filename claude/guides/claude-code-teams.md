### I built a web game with over 25K plays so far using Claude: You try to time the market against a couch that buys once and does nothing.

開発者が Claude を使って、市場のタイミングを計る能力を試すウェブゲーム「Beat the Couch」を構築し、25,000回以上プレイされている。プレイヤーは1万ドルでランダムな2年間のS&P 500実データで取引し、初日に買って何もしないソファと対決する。Claude がコード構造、特殊ケース処理、トーンの調整を支援し、フレームワーク不要の300KBの単一HTMLファイルとして実装された。統計的に有意な勝利かを判定するため、1,000回の順列テストも組み込まれている。
