### Claude Code tips for terminal users (from a senior dev)

Linux開発者による Claude Code のターミナル利用の実践的なTipsを紹介。6ヶ月以上の使用経験から、見落としがちな5つの隠れコマンドを解説。/statusline でステータスバーをカスタマイズ、! でシェルコマンド実行、@ でファイルパス補完、/add-dir で複数ディレクトリ追加、/btw で並行会話が可能。これらの機能により日常のワークフローが大幅に効率化される。
