### Claude Desktop・cowork のConnectorsの申請をしてみた（ローカル動作確認方法付き）

Claude Desktop の Connectors 申請手順を日本語で解説。MCP を Desktop App で使えるようにする仕組みで、申請には HTTPS 必須、プライバシーポリシー・ドキュメント公開、全ツールへの readOnlyHint/destructiveHint 追加が必要。OAuth 2.0 は動的クライアント（CIMD 推奨）、SSE ではなく Streamable HTTP での実装が推奨される。審査期間は不明で進捗確認もできない。
