### Claude Code Skillの「Eval/Gotchas」で使うほど賢くなる自己改善ループを作る

Claude Codeの~/.claude/skills/配下のSKILL.mdファイルに「Eval」と「Gotchas」セクションを必須化することで、スキルの品質劣化を防ぐ自己改善ループを構築する手法。Evalセクションで完了基準を明文化し自己採点を強制、Gotchasセクションで失敗パターンを記録することで、使うほど賢くなるスキル運用を実現。skills-usage.jsonに履歴を蓄積し、低品質スキルの発見や棚卸しに活用する。
