# Team Plan Migration

---

## 2026-03-29

### Claude CodeをAWS Bedrock経由からTeam planに移行した話

ツクリンクではClaude CodeをAWS Bedrock経由で利用していたが、1人あたり月$150のコストがTeam plan（Premium $125、Standard $30）を上回ったため移行を決定。当初はAWS SSOとの連携や少額請求のメリットでBedrock経由を選択していたが、Cowork機能の登場や非エンジニアからのニーズ増加により、Team planの方が総合的に有利と判断した。組織でClaude Codeを使う3つの経路（AWS Bedrock等のAPI、Anthropic API、Team plan）の比較も紹介している。
