### Claude Code + Playwright MCPで始めるレスポンシブ確認の自動化

Claude Code と Playwright MCP を組み合わせて、複数ブラウザ（Chromium・Firefox）で複数画面サイズ（PC・タブレット・スマホ）のレスポンシブ確認を自動化する手法を解説。自然言語でのプロンプト指示により、手動での繰り返し作業を効率化し、URLを引数に渡すだけで動作する汎用スクリプトを構築した実践事例。
