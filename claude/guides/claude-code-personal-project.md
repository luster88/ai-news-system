### HTML1ファイルでLinktree代替を自作した話【Claude Code × 個人開発】

Claude Codeを使ってLinktreeの代替となるHTML1ファイル完結型のリンクまとめページ生成ツールを1〜2時間で開発。ツール自体も生成されるリンクページもスタンドアロンで動作し、外部依存ゼロ。BOOTHで980円の買い切り型として販売中。Linktreeの月額課金（年間約1万円）と比べてコスパが良く、プライバシー面でも優位。
