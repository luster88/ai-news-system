### 【Python×Claude】金融データは自分で管理。3日で家計簿アプリ「MoneyRebirth」を作った話

MoneyForwardのバックアップCSVや各金融機関のデータを一元管理できる家計簿アプリ「MoneyRebirth」を、Claudeの支援を受けて3日間（実質5-6時間）で開発した事例。三菱UFJ銀行、三井住友カード、SBI証券、SuicaなどのデータをStreamlitベースで可視化し、ローカルまたはクラウドで動作。Claude ChatとClaude Codeの使い分けにより、開発サイクルを高速化した個人開発の実践記録。

### HTML1ファイルでLinktree代替を自作した話【Claude Code × 個人開発】

Claude Codeを使ってLinktreeの代替となるHTML1ファイル完結型のリンクまとめページ生成ツールを1〜2時間で開発。ツール自体も生成されるリンクページもスタンドアロンで動作し、外部依存ゼロ。BOOTHで980円の買い切り型として販売中。Linktreeの月額課金（年間約1万円）と比べてコスパが良く、プライバシー面でも優位。
