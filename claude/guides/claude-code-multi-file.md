## 2026-06-18

### 【Claude Code活用】複数ファイル跨ぎの修正を任せる

Claude Code で複数ファイルを同時に修正する際のベストプラクティスを紹介。実装指示書に「作成・更新ファイル一覧」を明示することで変更漏れを防ぎ、Claude Code は既存コードを読んで依存関係を自律的に把握して対応する。ImportError などのエラーも自己修正可能。3〜4ファイルの同時変更では指示書の価値が高い。
