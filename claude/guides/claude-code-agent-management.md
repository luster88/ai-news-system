### Claude Codeのサブエージェントが「だいたい完成」で帰ってくる問題と、委譲プロンプトのテンプレート

Claude Codeのサブエージェント（Agent tool）が検証せずに「完了しました」と早期終了する問題に対し、委譲プロンプトに含めるべき5項目のテンプレート（完了条件の具体化、検証ログ出力の義務化、エラー即報告など）と、エージェント定義ファイル側の自己点検チェックリストを実例とともに提示。委譲に適したタスクは大量読み込み・独立検証・並列バッチの3型に限定し、実装や設計判断はメインセッション直営が効率的だと運用経験から結論。
