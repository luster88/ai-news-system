### Claude Codeで仕様書をMarkdown/HTMLで記述を比較

Claude Code を使用した仕様書生成において、Markdown と HTML の比較検証を実施。HTML + Interactive 要素を使用すると視認性と操作性が大幅に向上するが、生成トークン数が2倍に増加し、仕様のメンテナンスコストが現実的でなくなる課題を指摘。図表現には mermaid や Playwright による自己検証も組み合わせて評価。
