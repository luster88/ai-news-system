## 2026-04-17

### Claude Code Learning Mode 完全ガイド — /output-style と TODO(human) の実務運用

Claude Code の Learning Mode は、AI がコードを全て書くのではなく、TODO(human) マーカーで意思決定ポイントを開発者に委譲する機能。/output-style と /config の両方で設定可能で、設定はチームで共有できる。Explanatory Mode との違いや、公式ドキュメントと GitHub の記述の矛盾についても詳しく解説されている。
