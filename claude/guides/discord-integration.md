### 【Claude Code】DiscordからClaude Codeを使う「Channels」

Claude Code v2.1.80以降で利用可能なDiscord連携プラグイン「Channels」の設定方法を解説。Discord Developer Portalでボットを作成し、トークンを取得してClaude Codeと連携することで、DiscordのDMやサーバーチャンネルからClaude Codeを操作できるようになる。Bunのインストールが必要で、ペアリングコードによる認証とポリシー設定でセキュアに利用可能。
