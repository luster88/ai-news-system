### MCPとは何か：AIに手足を与えるプロトコルの仕組みと実践ガイド2026

Anthropicが開発したModel Context Protocol（MCP）は、AIアプリケーションと外部ツール・データソースを統一的に接続するオープン標準プロトコルです。2026年4月時点で10,000を超える公開サーバーが存在し、Claude Code、Cursor、VS Code Copilot、ChatGPTなど主要AIツールが対応しています。MCPは「AIのUSB-C」と呼ばれ、M×N問題を解決することで、AIエージェントがテキスト回答を超えて実際の世界に働きかける基盤インフラとなっています。Tools・Resources・Promptsの3つのプリミティブを持ち、JSON-RPC 2.0ベースのステートフルプロトコルとして設計されています。
