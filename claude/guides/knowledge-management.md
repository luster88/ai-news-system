### AIフィードバックを「資産化」する3層構造：feedback_*.md → feedback-index → MEMORY.md

Claude Code や Cursor などの AI エージェントと長期運用する際、フィードバックを効率的に管理するための 3 層構造（feedback_*.md / feedback-index.md / MEMORY.md）を提案。詳細は個別ファイルに記録し、インデックスのみを常時ロードすることでコンテキスト窓を節約しつつ、知識を無制限に蓄積できる。Why と How to apply を明記し、frontmatter で構造化することで AI が推論可能な形式を維持する。
