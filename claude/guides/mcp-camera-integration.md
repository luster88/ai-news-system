## 2026-05-15

### 積んでた Wi-Fi カメラで Claude にようやく視覚を持たせれた

TP-Link の Wi-Fi カメラ（Tapo C220）を MCP サーバー経由で Claude に接続し、視覚機能を持たせる実装例。ONVIF 対応カメラを使い、Claude が自然言語でカメラを操作して画像を取得・認識できる。embodied-claude リポジトリのコードを参考に、セットアップから動作確認まで実施。バッテリー駆動モデルは ONVIF 非対応のため AC 電源が必要な点に注意。
