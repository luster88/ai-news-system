# Claude Code Dataviz

---

## 2026-07-03

### Claude Codeの dataviz スキルが凄かった

Claude Code 2.1.199に同梱されている/datavizスキルを使った売上ダッシュボード作成事例。色選びを最後に行う7ステップの手順に従うだけで、明度・彩度・色弱対応・コントラストが検証済みの8色パレットを使用したプロフェッショナルなダッシュボードが作成できる。ライト/ダークモード用に別色が用意され、よくあるアンチパターン（虹色バー、2軸重ね、過剰ラベル）を回避できる。
