### The Illusion of Finished Work in Claude Code

Claude Code のエージェント機能が生成する「完成したように見える」出力の問題について議論。クリーンな diff や成功したテストがあっても、実際の動作、セキュリティ、アーキテクチャ制約、エッジケースを見落とす可能性がある。エージェントが停止したことと、安全にマージできることは別問題。より良いレビュー機能（タスク、プラン、変更ファイル、コマンド実行履歴、未検証項目の表示など）の必要性を提起している。
