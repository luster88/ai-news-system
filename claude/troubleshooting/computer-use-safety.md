# Computer Use Safety

---

## 2026-03-30

### Don’t let Claude use your actual computer from the CLI

Claude の computer use 機能をローカル PC で直接実行することのリスクについて警告する投稿。AI エージェントは予測不能な失敗をするため、パスワードや API キーなどの重要データが保存された実機ではなく、サンドボックス化された仮想環境で実行すべきだと提言している。タスク完了後にリセットや破棄が可能な隔離環境の利用を推奨。
