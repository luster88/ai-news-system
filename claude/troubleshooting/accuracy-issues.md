### 4大AIに「同じ日の六曜」を3週連続で聞いたら毎週違う答えが返ってきた — Shirabe Text API 5/18 リリース告知

4大AI（ChatGPT/Claude/Perplexity/Gemini）に同じ日の六曜を3週連続で質問したところ、毎週異なる回答が返ってきた。AIは自然言語処理は得意だが、日本語固有の正確なデータ（六曜・形態素解析・住所など）の扱いが不安定。これを解決するため、2026年5月18日にShirabe Text APIがリリースされ、日本語テキスト処理のcanonical answerを提供する。暦API・住所APIに続く3本目のAPIで、Free枠あり、OpenAPI公開、GPTs/Claude Tool Use/LangChainから統合可能。
