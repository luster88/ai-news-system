## 2026-06-29

### Claude in Microsoft Foundry is now generally available.

Claude が Microsoft Foundry で一般提供開始されました。Azure 上でホストされ、Anthropic が運用します。Azure 顧客は Claude Opus 4.8 と Claude Haiku 4.5 を使用して、既存の Azure 環境内で本番アプリケーションやエージェントを構築できます。認証、課金、ガバナンスは既存の Azure の仕組みを利用でき、対象顧客は Microsoft Azure Consumption Commitment から利用料を引き落とせます。プロンプトキャッシングと拡張思考が現在サポートされており、今後さらに機能が追加される予定です。
