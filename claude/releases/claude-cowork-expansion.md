# Claude Cowork Expansion

## 2026-07-07

### Anthropic's Claude Cowork AI agent is now available on mobile and web

Anthropic が Claude Cowork をモバイルとウェブに展開開始。従来はデスクトップアプリ限定だったが、Max サブスクライバーから段階的にベータ版を公開。デバイス間でタスクを継続でき、バックグラウンドで動作継続。利用の90%以上がソフトウェア開発以外で、ビジネス運用とコンテンツ作成が約半分を占める。ローカルファイルアクセスやComputer Useなどはデスクトップアプリが引き続き必要。

## 2026-05-13

### Anthropic launches Claude for Small Business to embed AI into the tools you forgot you pay for

Anthropic が中小企業向けに「Claude for Small Business」を発表。QuickBooks、PayPal、HubSpot、Canva などの既存ビジネスツールに Claude を統合し、財務、営業、マーケティングなどの 15 種類のワークフローを提供。中小企業の AI 導入遅れを解消し、日常業務を自動化することを目指す。全米でワークショップツアーも実施予定。

### Anthropic courts a new kind of customer: small business owners

Anthropic が中小企業向けの新サービス「Claude for Small Business」を発表。Claude Cowork 内のトグル機能で、簿記、ビジネスインサイト、広告キャンペーン生成などの自動化サービスを提供。QuickBooks、Canva、DocuSign、HubSpot、PayPal などとの統合も実装され、これまで大企業中心だった AI 導入を中小企業層へ拡大する戦略。

## 2026-04-09

### Claude Cowork expands to all paid plans on macOS and Windows with new org controls

Claude Cowork が macOS と Windows のすべての有料プランで利用可能になりました。組織管理機能として、役割ベースのアクセス制御、チーム別予算制限、利用分析、OpenTelemetry モニタリングが追加されました。新しい Zoom コネクターにより会議の要約やタスクを直接取り込めます。Cowork は Claude Code の非開発者向けバージョンで、ローカルファイルへのアクセスが可能です。
