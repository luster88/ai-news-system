# Computer Control

---

## 2026-04-03

### Claude Code and Cowork now let Anthropic's AI take control of your Mac or Windows desktop

Anthropic が Claude Cowork と Claude Code に、Mac と Windows のデスクトップを AI が直接操作できる新機能を追加しました。この機能は Vercept AI の買収により実現し、アプリ起動、ブラウザ操作、スプレッドシート入力などが可能です。現在は研究プレビュー版として macOS で提供され、Pro および Max ユーザーが Windows アプリでも利用できます。また、リモートでコンピューターを制御できる「Dispatch」機能も同時にリリースされました。
