## 2026-06-02

### 【6/15〜】Claude Code の課金変更、結局「何が対象で何が対象外」なのか公式情報だけで整理した

2026年6月15日から、Claude Agent SDKやclaude -p、GitHub Actionsなどプログラム経由の利用が月次クレジット制に移行します。ターミナルやIDEでの対話的なClaude Code利用は対象外で、従来通りサブスクリプションで使用可能です。Max Plusは$150、Proは$50の月次クレジットが付与され、超過分はusage creditsを有効化している場合のみAPI料金で課金されます。クレジットは翌月繰越不可で個人アカウント単位での管理となります。

## 2026-04-22

### Claude ProでClaude Codeが使えなくなった——新プラン比較と移行の判断フロー

2026年4月、AnthropicはClaude ProプランからClaude Code（ターミナルCLI）を除外し、Max 5x以上（月$100〜）でのみ利用可能に変更。Proプラン自体は月$20で継続しているが、Claude Codeを使うには実質5倍の料金が必要となった。背景にはサードパーティツールによるプロンプトキャッシュのバイパスや計算リソースの過大消費があり、4月4日にはOpenClaw等のツール経由利用も別途課金に移行。エンジニア向けに各プランの機能比較と移行判断フローが整理されている。
