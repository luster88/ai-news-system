# Managed Agents Beta

---

## 2026-05-19

### Self-hosted sandboxes and MCP tunnels for Claude Managed Agents are now in public beta.

Claude Managed Agents の新機能として、セルフホスト型サンドボックスと MCP トンネルがパブリックベータで公開されました。セルフホスト型サンドボックスにより、Cloudflare、Daytona、Modal、Vercel などのプロバイダーや自社インフラ上でエージェントを実行できます。MCP トンネルは、プライベートネットワーク内の MCP サーバーをインターネットに公開せずにエージェントと接続可能にします。Claude Platform で本日から利用可能です。
