### KIOKU v0.7.0 — Codex CLI / Gemini CLI にも対応

KIOKU v0.7.0 がリリースされ、Codex CLI / Gemini CLI でも自動セッションログ機能が利用可能になった。v0.6 ではスキル共有のみだったマルチエージェント対応が、今回のアップデートで記憶も共有される仕組みに進化。591 行のモノリシックなスクリプトを core + 3 adapter 構造にリファクタリングし、Claude Code 固有だった Hook システムを各エージェントに対応させた。セキュリティ実装の強化やビジュアライザー機能も追加された。
