## 2026-04-21

### Claude Cowork can now build live artifacts

Claude Cowork に新機能「Live Artifacts」が追加されました。アプリやファイルに接続されたダッシュボードやトラッカーをリアルタイムで構築でき、開くたびに最新データで更新されます。作成した全てのアーティファクトは新しい「Live Artifacts」タブに保存され、バージョン履歴も管理できます。すべての有料プランで本日から利用可能です。
