### コーディングエージェントが人間承認なしでpush・ドラフトPRまで到達する時代に：Claude Code v2.1.198・Cursor 3.8・GitHub Desktop 3.6で見直すブランチ保護とトリガー設計

2026年7月リリースのClaude Code v2.1.198がバックグラウンドエージェントの既定動作を変更し、人間の承認なしにcommit・push・ドラフトPR作成まで自律実行するようになった。Cursor 3.8もイベント駆動Automationsを導入し、エージェント自律化が加速。この変化により、ブランチ保護とCI/CDトリガー設計がエンジニアの唯一の安全弁となり、ガバナンス設計の見直しが必須となっている。
