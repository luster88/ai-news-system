## 2026-05-31

### Anthropic「Project Glasswing」初動レポート――1か月で1万件超の脆弱性発見、Mythosが"結果"を出し始めた

Anthropicの防御的サイバーセキュリティプログラム「Project Glasswing」が、Claude Mythosモデルを活用して開始1か月で1万件超の脆弱性を発見。wolfSSLでの証明書偽造可能な脆弱性検出や150万ドル相当の銀行詐欺阻止など、具体的な成果を報告。Claude Securityのパブリックベータ開始とCyber Verification Programの新設により、AIによるセキュリティ監査が実用段階に入った。

## 2026-05-01

### Anthropic launches Claude Security to give defenders the same AI edge attackers already have

Anthropic が Claude Enterprise 顧客向けに Claude Security のパブリックベータを開始。Claude Opus 4.7 を使用してコードの脆弱性をスキャンし、パッチを推奨する。CrowdStrike、Palo Alto Networks、SentinelOne、Wiz などのパートナーが Opus 4.7 を自社のセキュリティ製品に統合中。攻撃者と同じ AI の優位性を防御側にも提供することを目指している。
