# Claude For Education

---

## 2026-07-14

### Anthropic opens Claude for Teachers with a promise not to train models on student data

Anthropic が米国の K-12 教育者向けに Claude for Teachers を無償提供開始。Claude Code と Cowork エージェント機能、教育向けスキルライブラリ、全 50 州のカリキュラム対応コンテンツを含む。学生データはモデル訓練に使用しないことを明言し、アメリカ教員連盟とプライバシー基準を策定中。Canva Education や MagicSchool などとの統合も提供し、デトロイトの公立学校でパイロットプログラムを実施予定。
