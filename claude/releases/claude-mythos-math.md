### Claude Mythos reportedly solves OpenAI's landmark Erdős problem with a "cute, simple proof"

Anthropic の Claude Mythos が OpenAI の画期的な数学的成果である Erdős 単位距離予想の反証問題を「シンプルで美しい証明」で解決したと報告されました。Anthropic のエンジニアによると、Mythos は OpenAI のモデルとは異なるアプローチを取り、複数の独立した Claude Code インスタンスを使ったテストシステムで成果を上げました。Google DeepMind も最近 AI 支援システムで9つの Erdős 問題を解決しましたが、形式証明言語 Lean を使用している点で異なります。
