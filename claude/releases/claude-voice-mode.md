## 2026-07-23

### Anthropic updates Claude voice mode with more capable models

Anthropic が Claude の音声モードを大幅にアップデート。Opus、Sonnet、Haiku の各モデルを選択可能になり、Gmail や Slack などの外部アプリ連携も実現。OpenAI の音声モードと異なり、ツール連携により実作業が可能だが、音声モデル自体の改善はなし。多言語対応はベータ版で提供中。
