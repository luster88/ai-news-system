# Claude Desktop Linux

---

## 2026-06-30

### Claude Desktop is now available on Linux (Ubuntu and Debian) in beta.

Claude Desktop が Ubuntu と Debian の Linux 環境でベータ版として利用可能になりました。有料プランユーザーは Claude Code、Claude Cowork、Chat 機能を利用できます。Computer Use 機能は今回のリリースには含まれていませんが、今後追加予定です。claude.com/download からダウンロード可能で、数十万人の Linux ユーザーが待ち望んでいた公式デスクトップ体験が提供されます。
