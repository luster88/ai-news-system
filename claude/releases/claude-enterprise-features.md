# Claude Enterprise Features

---

## 2026-06-23

### Anthropic’s Claude Tag is learning your company, one Slack message at a time

Anthropic が Slack 内で常駐する新サービス「Claude Tag」をベータ版として Enterprise/Team 顧客向けに提供開始。チャンネル内で @Claude をタグ付けすることで、継続的なコンテキストとメモリを持ち、タスクを自動分解して実行。アンビエントモードでは自発的に会話に参加し、組織全体の情報を把握しながらチーム作業をサポートする。各チャンネルごとに独立した Claude アイデンティティが設定され、管理者が権限とアクセス範囲を制御可能。
