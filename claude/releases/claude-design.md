### Introducing Claude Design by Anthropic Labs

Anthropic Labs が Claude Design を発表。Claude Opus 4.7 を搭載した新しいデザインツールで、会話を通じてデザイン、プロトタイプ、スライド、資料を作成できる。コードベースやデザインファイルを読み込んでチームのデザインシステムを構築し、自動的に適用する機能を持つ。Canva、PDF、PPTX へのエクスポートや Claude Code への引き継ぎが可能。Pro、Max、Team、Enterprise プランで研究プレビューとして順次提供開始。
