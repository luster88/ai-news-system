## 2026-05-19

### Cloudflare says Anthropic's Mythos Preview finds exploit chains that earlier frontier models missed

CloudflareがAnthropicのセキュリティ特化AIモデル「Mythos Preview」を50以上の自社リポジトリでテスト。このモデルは複数の小さな脆弱性を連鎖させて実際に悪用可能なエクスプロイトチェーンを構築し、PoC（概念実証）コードを自動で記述・コンパイル・実行して検証できる。従来のフロンティアモデルは個別のバグ検出は可能だったが、脆弱性の連鎖や実際の悪用可能性の証明には至らなかった。Cloudflareは最大50の並列エージェントと敵対的レビューを組み合わせた多段階システムを構築し、攻撃者も同じ能力を利用できると警告している。
