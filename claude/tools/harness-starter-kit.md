### Harness Starter Kit が Codex と Claude Code に対応しました

Harness Starter Kitが、CodexとClaude Codeに対応し、Agent Skills/pluginを通じて関連ワークフローを利用可能になりました。このツールキットは、AI coding agentがプロジェクト内で繰り返し発生する問題を長期的に維持できるリポジトリのルール、チェック、失敗記録として残すためのprompt-firstなオープンソースツールです。/harness doctorや/harness reviewなどのワークフローを通じて、agent向けの指示やプロジェクト制約、フィードバックループを確認でき、Python、TypeScript、React、Go、Rustなど複数の技術スタック向けのプロファイルが提供されています。
