### Turned a desk lamp into a Claude Code status indicator

Claude Codeの動作状況を可視化する物理的なステータスインジケーターの実装例。Bluetooth対応デスクランプをClaude Codeのフックと連携させ、Claudeが作業中は青色のスピンアニメーション、ユーザー入力待ちはピンク、アイドル時は暖色の白に光るよう制御している。オープンソースプロジェクトを基にしたPythonスクリプトでBLE経由で制御し、今後Philips Hueスマート電球との統合も計画中。
