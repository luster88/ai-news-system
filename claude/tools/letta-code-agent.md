# Letta Code Agent

---

## 2026-07-09

### Letta Code徹底解説｜記憶を持つステートフルAIエージェントCLIの仕組みと使い方

Letta Codeは記憶とアイデンティティを持つステートフルなAIエージェントCLIツール。セッションをまたいで文脈を保持し、メモリブロック・MemFS・スキルの多層で記憶を管理する。Claude CodeやCodex CLIが「セッション単位の道具」であるのに対し、Letta Codeは「使うほど賢くなる同僚」を目指す設計。エージェント自身が記憶・スキル・ハーネスを書き換えながら成長する点が特徴。
