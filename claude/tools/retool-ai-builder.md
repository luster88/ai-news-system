### Retool新AIアプリビルダー（R2）徹底解説｜React+TypeScript生成とエンタープライズガバナンス

Retool が新AIアプリビルダー（R2 / Retool 4.0）を公開。自然言語からReact 19 + TypeScript で本番データ接続の業務アプリを生成する。生成基盤は r2 agent / agent sandbox / MCP サーバーで構成され、MCP経由でコーディングエージェントからも構築可能。核心はガバナンス機能で、認証・RBAC・監査をプラットフォームレベルで一元管理する。OSSではなく商用SaaS（Retool Cloud + セルフホスト）として提供される。
