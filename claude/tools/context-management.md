### Claude Code を複数プロジェクトで本気で使うための「AI文脈オーケストレーター」ツールを自作した

複数のプロジェクトを並行して進める際、Claude Code に渡す文脈管理のコストが課題となることから、Windows 用のシステムトレイ常駐ツール「Curia」を自作した記事。プロジェクトごとの文脈レイヤー管理、LLM Wiki による知識の永続化、案件別 AI 振る舞い切り替えを実現し、ホットキーでの素早い呼び出しや Dashboard での状況把握、Quick Capture による記録、Agent Hub でのルール管理など実用的な機能を備える。Andrej Karpathy の「LLM Wiki」の考え方を取り入れ、使い捨てではない知識資産の蓄積を重視している。
