### Claude Codeで「1プロンプトサイト複製」が話題だけど、本当にヤバいのは“UI実装の重心”がズレ始めたこと

Claude CodeのChrome連携とスキルシステムにより、WebサイトのクローンがスクリーンショットではなくgetComputedStyle()などブラウザの実際の計算値を基に再構成できるようになった。これは「AIがUIを推測する」段階から「ブラウザを開発ツールとして使う」段階への移行を意味し、Next.js/React/Tailwindベースでの自動再現が可能に。重要なのは他人のサイトの複製ではなく、自分が権利を持つLPや過去案件を土台化し、編集判断や体験設計に集中できる点。
