### Claude Codeのカスタムコマンドで転職活動の応募書類を自動生成するツールを作ってみた

Claude Codeのカスタムスラッシュコマンド機能を使い、転職活動の応募書類（志望動機・自己PR・職務要約）を企業ごとに自動生成するツール「job-hunting-kit」の開発事例。キャリア情報をmy-career.mdとして永続化し、/setupでヒアリング、/applyで企業名を渡すだけで応募書類を一括生成できる。ChatGPT/Claude Web版のコピペ運用と異なり、情報の再入力不要・フォーマット統一・ローカルファイル管理が可能。
