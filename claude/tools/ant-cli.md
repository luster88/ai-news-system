### ant CLI入門 — Claude APIをターミナルとCI/CDで使いこなす完全ガイド

Anthropic が公式リリースした ant CLI（v1.1.0）の完全ガイド。Claude API をターミナルから操作できる Go 製 CLI ツールで、typed flags・YAML heredoc による直感的なリクエスト作成、GJSON による柔軟なレスポンス整形、Managed Agents の管理機能を提供。YAML 定義ファイルを Git 管理して CI/CD パイプラインで API リソースを同期でき、Claude Code とのネイティブ連携も可能。
