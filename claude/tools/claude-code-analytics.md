### Claude Codeの利用状況をチームで可視化するダッシュボードをOSSで公開しました

Claude Codeのスキル、MCPサーバー、サブエージェントの利用状況とトークン消費量を自動収集・可視化するOSSダッシュボードが公開されました。Stop hookとPluginを活用し、セッション終了時にトランスクリプトを自動解析してAPIに送信。チーム全体の活用度を把握し、未活用機能の発見や利用促進に貢献します。公開後1週間でサブエージェント利用が1.5倍増加した事例に触発されて開発されました。
