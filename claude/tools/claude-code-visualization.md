### I got tired of watching Claude Code work in a plain terminal so I built it a room.

Claude Code の作業を視覚化する 3D オフィス環境を開発したユーザー事例。エージェントをロボットとして表示し、ターミナルの作業内容をリアルタイムで 3D 空間上に反映。プロジェクトの 90% 以上を Claude が担当し、Web アプリから Electron への移行提案も Claude 自身が行った。複数エージェントをハイブとして管理し、ファイル操作履歴を追跡可能。

### claudemap徹底解説：AIが書いたコードを概念地図で可視化するツール

## 2026-06-27

### Claude Codeのセッションを見下ろし型オフィスとして可視化するツールを作った

Claude Codeのセッションを見下ろし型オフィスとして可視化するmacOS専用デスクトップアプリ「claude-code-park」が公開されました。~/.claude/を監視し、エージェント・hooks・スキルをリアルタイムで可視化します。アプリは100% Claude Codeで開発され、外部通信なしで動作する点が特徴です。開発中、Claudeに「散歩」を促すことで行き詰まりを打開したエピソードも紹介されています。
