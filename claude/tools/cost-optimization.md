### Open-source tool pxpipe hides text in PNGs to cut Claude Code and Fable 5 token costs up to 70%

オープンソースツール pxpipe が、Claude Code や Fable 5 のトークンコストを最大70%削減する技術を提供。テキストを高密度な PNG 画像に変換することで、画像トークンの固定価格を利用してコストを大幅に削減。システムプロンプトやツールドキュメントなど静的コンテンツを画像化し、48,000文字を約2,700トークンに圧縮。ただし、正確性の低下や処理速度の低下というトレードオフがある。
