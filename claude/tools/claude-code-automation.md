### I built an AI job search system with Claude Code that scored 740+ offers and landed me a job. Just open sourced it.

Reddit ユーザーが Claude Code を使用して構築した AI 求職支援システムをオープンソース化。740件以上の求人情報を評価し、求人ごとにカスタマイズされた履歴書 PDF を生成、面接準備やフォーム入力支援まで行う。14種類のスキルモード、Go 製のターミナルダッシュボード、45社以上の企業設定を含み、CLAUDE.md で定義されたスキルを Claude Code が実行する実践的なシステム。
