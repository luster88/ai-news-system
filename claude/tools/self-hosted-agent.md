# Self Hosted Agent

---

## 2026-07-05

### netclode徹底解説：スマホから使える“自前ホスト型”コーディングエージェント基盤

netclodeは、コーディングエージェントを自前のインフラで動かし、iOS/macOSアプリから操作できるセルフホスト型の基盤です。Kata ContainersとCloud HypervisorのmicroVMで各セッションを隔離し、スマホから安全にClaude Code環境を利用できます。クラウド型サービスの「コードを預ける」「UXを選べない」という課題を解決し、Ollama対応やGitHub統合も備えています。導入は上級者向けですが、データ主権とモバイル体験を両立させた個人開発OSS（約186スター）です。
