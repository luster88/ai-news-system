### AIエージェント×GitHub連携：トークン効率で選ぶなら「ghコマンド」か「MCP」か？

GitHub連携でAIエージェントを使う際、トークン効率の観点から「ghコマンド」と「MCP」を比較。ghコマンドはjq/xargsでデータを絞り込めるため、MCPよりトークン消費を抑えられる。MCPは環境構築不要で安全性が高いが、システムプロンプトで初期トークンを消費する点がデメリット。従量課金時代を見据えたトークン節約の実践的な検討記事。
