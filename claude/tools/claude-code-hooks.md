# Claude Code Hooks

## 2026-04-11

### Claude Codeのhook、どれを入れればいい？——5つの質問で最適な安全設定が分かるツールを作った

Claude Code の 650 種類以上ある hook から最適なものを選ぶための Hook Selector ツールが公開された。5つの質問に答えるだけで、ファイル削除・トークン消費・秘密漏洩・コード品質など懸念事項に応じた推奨 hook セットが提示される。800時間以上の自律運用経験から16個の hook データベースを構築し、使い方パターンに基づいてスコアリングする仕組み。推薦された hook は npx cc-safe-setup コマンドで簡単にインストール可能。
