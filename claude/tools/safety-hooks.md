### Destructive Command Guard（dcg）とは｜AIエージェントの危険コマンドを実行前ブロック

Destructive Command Guard（dcg）は、AIコーディングエージェントが危険なシェルコマンド（git reset --hard、rm -rfなど）を実行する前に検出してブロックするRust製フックツール。Claude Code、Cursor、Copilotなど幅広いエージェントに対応し、90超のパックでgit/docker/k8s/DBなどを網羅。ライセンスは「MIT + OpenAI/Anthropic Rider」で、OpenAI・Anthropicには学習・評価含め一切の権利を与えない独自条項が付く。fail-open設計のため完全な防壁ではなく、v0.6.x（pre-1.0）段階の実質1人開発プロジェクト。
