### Clawdmeter turns your Claude Code usage stats into a tiny desktop dashboard

オープンソースプロジェクト「Clawdmeter」が登場。Claude Code の使用統計をデスクトップ上の小型ハードウェアダッシュボードで可視化できるツール。アイスランドの開発者が Claude の支援を受けて数日で開発し、AI トークン消費を最大化する「tokenmaxxing」トレンドを象徴するプロジェクトとして注目されている。ESP32ベースのデバイスで Bluetooth 接続により動作する。

## 2026-04-24

### Claude / Codex / ZAI / Ollama の使用量を一画面でまとめて見る「aimo」を作った

Claude、ChatGPT、Z.ai、Ollamaの使用量を1画面で確認できるダッシュボードツール「aimo」が公開されました。ブラウザ拡張とローカルHTTPサーバーで動作し、各サービスのログインセッションを活用するため追加のAPI key設定が不要です。複数AIサービスの利用制限を統合表示し、エージェント向けJSON APIも提供します。
