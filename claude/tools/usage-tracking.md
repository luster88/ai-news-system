# Usage Tracking

---

## 2026-03-28

### ccreport を作った

Claude CodeとCodexの使用量を時系列で可視化・分析するツール「ccreport」が公開された。既存のccusageがCLIでの都度確認型であるのに対し、ccreportはクラウドに使用データを蓄積してヒートマップで可視化し、月次・年次の振り返りや複数ツールの統合管理を可能にする。ローカルのセッションデータを自動送信し、トークン消費量やコストを一元管理できる。
