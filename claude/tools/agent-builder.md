### Open Agent Builder徹底解説｜ノーコードでAIエージェントを構築するFirecrawl製OSS

FirecrawlによるOSS「Open Agent Builder」はドラッグ&ドロップでAIエージェントのワークフローを構築できるノーコードプラットフォーム。8種類のノード（Start/Agent/MCP Tools/Transform/If-Else/While/User Approval/End）でワークフローを可視化し、Claude/GPT-5/Groqに対応。LangGraph実行エンジンでリアルタイム実行とhuman-in-the-loop承認機能を備え、MCP連携でFirecrawl等の外部サービスと統合可能。MITライセンスで公開されており、非エンジニアでもAIエージェント設計に参画できる点が特徴。
