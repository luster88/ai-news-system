### memory-os解説｜AIエージェントの記憶をローカル完結でOS的に階層管理する7層メモリ基盤OSS

memory-osは、AIエージェント「Hermes Agent」向けのローカル完結型記憶基盤OSSです。7層構造で記憶を管理し、最大の特徴はLayer 7「Ground Truth階層」で、注入された記憶をエージェントに確実に使わせる識別レイヤーを持ちます。Qdrant・Redis・ARQ WorkerをローカルPC上で動かし、外部SaaSに記憶を出さない設計で、約1,151スター、MITライセンスで公開されています。
