### I built an open-source canvas where Claude responds beside your handwritings

PenEcho は、手書きの数式や図の横に Claude が応答するオープンソースのキャンバスツール。物理・数学研究者向けに開発され、20,000×20,000 の論理キャンバス上で手書き入力後、関連領域を Claude に送信し、その場で編集可能な応答を得られる。ローカル実行で Anthropic API や Claude Code CLI、OpenAI 互換 API に対応し、Opus 4.8（特に fb5）の視覚理解能力を活用している。
