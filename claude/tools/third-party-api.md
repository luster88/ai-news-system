### Cherry Studio / Chatbox / Cursorで使えるOpenAI互換API「CVRIX」を公開しました

OpenAI互換API「CVRIX」が公開されました。企業契約や検証用の余剰API枠を活用することで、通常より約80%安い価格でGPT-5.5やClaude 4.8などのAIモデルを利用可能です。Cherry Studio、Chatbox、Cursor、Clineなど主要なAIツールに対応しており、Base URL、API Key、Modelの3つを設定するだけで使えます。毎日大量にAIを使う人向けのコスト削減ソリューションとして提供されています。
