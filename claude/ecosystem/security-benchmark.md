### New benchmark shows Claude Mythos and GPT-5.5 can develop real browser exploits autonomously

カーネギーメロン大学の研究チームが、AIエージェントがGoogle V8エンジンの脆弱性を自律的に悪用できる能力を測定する新ベンチマーク「ExploitBench」を開発。Anthropic の Claude Mythos Preview モデルは、OpenAI の GPT-5.5 を大きく上回り、熟練したセキュリティ研究者並みの性能を示した。Mythos は41件中21件で最高レベルの任意コード実行に到達したが、テストコストは約36,400ドルと GPT-5.5 の約12倍に達し、コスト効率に課題を残す結果となった。

### GPT-5.5 matches Claude Mythos in cyber attack tests, UK AI Security Institute finds

英国AI Security Instituteの評価により、OpenAIのGPT-5.5がサイバー攻撃テストでAnthropicのClaude Mythos Previewと同等の能力を示したことが判明。複雑な多段階企業攻撃シミュレーションを完全に解決した2番目のモデルとなり、単独の専門タスクではMythosを上回る結果も。これはAI全般の改善（自律性、推論、コーディング）の副産物としてサイバー攻撃能力が向上している広範なトレンドの証拠とされる。
