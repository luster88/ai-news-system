### Mistral CEO Arthur Mensch warns France against letting Anthropic's Mythos scan military code bases

Mistral AI の CEO Arthur Mensch 氏が、フランス軍のコードベースを Anthropic の AI モデル「Mythos」でスキャンすることに警告を発した。Mythos は攻撃の orchestration、脆弱性検出、エクスプロイト提案が可能であり、軍事インフラへの依存関係を生むリスクがあると指摘。EU は現在 OpenAI や Anthropic とサイバーセキュリティモデルへの早期アクセスを交渉中。Mistral は米国投資家の保有比率を 30% 未満に抑え、独立性を維持する方針。
