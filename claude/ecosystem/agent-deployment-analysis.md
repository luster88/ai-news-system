### Anthropic just confirmed why 90% of non-coding AI agents fail in production

Anthropic が公開した分析によると、Claude API 上のエージェント活用の約50%がソフトウェアエンジニアリング分野に集中しており、営業・マーケティング・財務などは一桁台に留まっている。この偏りの原因は AI の能力不足ではなく、コード以外の業務データが非構造化で整理されていないことにある。コードは Git リポジトリで構造化され、明確なエラーメッセージが得られるのに対し、営業やマーケティングのデータは常に変化し、フォーマットも不統一である。非コーディングエージェントの失敗は、モデルの推論能力ではなく、データ品質の問題に起因することが多い。
