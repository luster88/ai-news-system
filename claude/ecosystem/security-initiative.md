## 2026-06-02

### Anthropic scales Project Glasswing to 150 partners across 15 countries to hunt critical software flaws

Anthropic が Project Glasswing を150の新パートナー組織（15カ国以上）に拡大。Claude Mythos Preview モデルを使用して重要ソフトウェアのセキュリティ脆弱性をスキャンし、既に10,000件以上の深刻な脆弱性を発見。エネルギー、医療、通信などの重要インフラ分野が対象で、EU のサイバーセキュリティ機関 ENISA もアクセス予定。Anthropic は商用製品 Claude Security も提供しており、Mythos クラスのモデルによる攻撃からの保護をビジネス化している。
