## 2026-06-17

### Glasswing外のすべてのシステムへ。「なぜ今、ローカルで動くセキュリティCLIを作ったか」

Anthropic が公開しなかった Mythos モデルはサイバー攻撃能力が極めて高く、攻撃者より先に防御側に渡すため Project Glasswing という主要インフラ企業向けプログラムが開始された。日本を含む外国籍ユーザーは Fable 5/Mythos 5 にアクセスできない状況で、Glasswing の外側にいる中小チームや個人が自分の責任範囲で使えるセキュリティ CLI「Carapace」を開発した背景と設計思想が語られている。サーバーレス・ローカル実行・ユーザー自身の API キー使用という設計により、コードを預けず低コストで脆弱性スキャンを実現する。
