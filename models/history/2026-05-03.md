# 最新AIモデルまとめ - 2026-05-03

## 概要
- 2026年5月時点で、OpenAIのGPT-5.5シリーズ（特にxhigh/high設定）が最高性能を記録し、Intelligence Index 59-60を達成。Claude Opus 4.7（max）が57で続く。
- Anthropicは4月にClaude Opus 4.7をリリースし、エージェント型コーディングで大幅な性能向上を実現。同時にClaude Design、Claude Cowork Projects/Dispatch機能を追加し、デスクトップAI統合を強化。
- GoogleはGemini 3.1 Pro Previewを投入し、Intelligence Index 57で上位に食い込む。価格は$4.50/1M tokensと競合より安価で、コスパに優れる。
- DeepSeek V4/V4 Proが4月末に発表され、1.6兆パラメータMoE、100万トークンコンテキスト、推論コスト73%削減を実現。中国製モデルの競争力が向上。
- 価格競争が激化し、DeepSeek V4 Flashは$0.18/1M tokens、Qwen3.6シリーズも$0.56-2.92/1M tokensと低価格。一方でOpenAI/Anthropicは高性能モデルで$10-11/1M tokensの価格帯を維持。

## 今週の新モデル・アップデート

### GPT-5.5 (OpenAI)
- 2026年4月24日リリース。最高性能モデルで、xhigh/high/medium/lowの推論レベルを選択可能。コーディング、研究、データ分析など複雑なタスクに対応。同時にCodexを統合し、専用コーディングモデルを廃止。
- 参照: [https://openai.com/index/introducing-gpt-5-5](https://openai.com/index/introducing-gpt-5-5)

### Claude Opus 4.7 (Anthropic)
- 2026年4月リリース。エージェント型コーディングで大幅な性能向上。Adaptive thinking機能を搭載し、100万トークンコンテキストをサポート。新tokenizerにより同じテキストが最大47%多くのトークンに分割されるため、実質コストは増加。
- 参照: [https://www.anthropic.com/news/claude-opus-4-7](https://www.anthropic.com/news/claude-opus-4-7)

### Claude Design (Anthropic)
- 2026年4月17日発表。Claude Opus 4.7ベースのビジュアル制作特化UI。自然言語からプロトタイプやプレゼンテーションを生成し、スライダーでリアルタイム編集可能。Figma対抗として位置づけられる。
- 参照: [https://claude.ai/design](https://claude.ai/design)

### Claude Cowork Projects & Dispatch (Anthropic)
- 2026年3月20日追加。Projectsはプロジェクトごとにファイル・指示・作業履歴をローカル保存。Dispatchは外出先からデスクトップのAIエージェントを操作可能にする機能。38以上のMCPコネクター（Gmail、Slack、Notion等）と連携。

### DeepSeek V4 / V4 Pro (DeepSeek)
- 2026年4月29日発表。1.6兆パラメータMoE、100万トークンコンテキスト、推論コスト73%削減を実現。華為昇腾950PRチップ（CANN）への移行により、中国初の完全自主算力基盤を持つ最先端AIモデルに。4月26日には入力キャッシュ価格を従来の1/10に値下げ。

### Gemini 3.1 Pro Preview (Google)
- 2026年4月リリース。最新の性能・知能・使いやすさの改善を実現。マルチモーダル理解、エージェント機能、vibe-codingに強み。価格は$2.00-4.00/1M tokens（入力）と競合より安価。

### Claude Mythos Preview (Anthropic)
- 2026年4月7日発表。サイバーセキュリティ特化の高度AIモデル。Project Glasswingの一環として約40組織に限定提供。攻撃能力が高すぎるため一般公開せず、防御目的に限定。NSAも脆弱性スキャンに使用中。

### Claude Sonnet 4.6 (Anthropic)
- 2026年2月17日リリース。Opusクラスの性能をSonnet価格で提供。コーディング、エージェント機能、長文推論、コンピュータ操作で大幅向上。100万トークンコンテキストをベータ提供。Extended thinking対応。
- 参照: [https://www.anthropic.com/news/claude-sonnet-4-6](https://www.anthropic.com/news/claude-sonnet-4-6)

### OpenAI Codex (AWS統合) (OpenAI)
- 2026年4月28日、OpenAIのGPTモデル、Codex、Managed AgentsがAWS（Amazon Bedrock）で利用可能に。Microsoft Azure独占契約終了後の展開。CLI、デスクトップアプリ、VS Code拡張から利用可能。
- 参照: [https://openai.com/index/openai-on-aws](https://openai.com/index/openai-on-aws)

### Gemini Enterprise Agent Platform (Google)
- 2026年4月22日発表。Vertex AIの後継として、AIエージェント組織活用の基盤を提供。Agent Studio（ローコード）、Agent Development Kit（コードベース）を含む。構築・展開・ガバナンス・最適化の4カテゴリでツール群を提供。

### xAI Custom Voices (xAI)
- 2026年5月2日発表。わずか1分間の音声録音から声をクローン化。2段階認証プロセスで悪用を防止。28言語対応、80以上のプリセット音声を含むVoice Libraryも提供。Grok Speech-to-Text/Text-to-Speech APIと連携。

### NEC × Anthropic 戦略提携 (NEC / Anthropic)
- 2026年4月23日発表。NECが日本企業初のAnthropicグローバルパートナーに。全グループ従業員約3万人にClaude Codeを展開。金融・製造・自治体向けにClaude Coworkベースの業界特化型ソリューションを共同開発予定。
