## 今週の新モデル・アップデート

### Claude Opus 4.7 (Anthropic)
- 2026年5月リリース。エージェントコーディングで大幅改善、Adaptive thinking搭載。1Mコンテキスト標準価格化（$5/25 per 1M tokens）。SWE-Bench等でClaude Opus 4.6を上回る性能を実証。
- 参照: [https://docs.anthropic.com/en/docs/about-claude/models](https://docs.anthropic.com/en/docs/about-claude/models)

### Gemini 3.5 Flash (Google)
- 2026年5月20日発表。コーディング・エージェント特化で、一部ベンチマークでGemini 3.1 Proを上回る。出力速度は他社フロンティアモデルの4倍。API価格は$1.50/9.00 per 1M tokens。
- 参照: [https://ai.google.dev/pricing](https://ai.google.dev/pricing)

### GPT-5.5 (OpenAI)
- 2026年5月時点で最高性能（Intelligence Index 60）。xhigh/high/medium/lowの4段階effort調整可能。GPT-Realtime-2など音声モデルも同時発表。922Kコンテキスト。
- 参照: [https://openai.com/index/introducing-gpt-5-4-mini-and-nano](https://openai.com/index/introducing-gpt-5-4-mini-and-nano)

### Codex (OpenAI) (OpenAI)
- 週間400万人以上が利用するコーディングエージェント。2026年5月にAppshots機能（Mac向け）、モバイルアプリ対応を追加。Dell、NVIDIA、Rampなど企業導入が加速。
- 参照: [https://openai.com/index/openai-to-acquire-astral](https://openai.com/index/openai-to-acquire-astral)

### Claude Code (Anthropic)
- 2026年4-5月にRoutines（クラウドスケジュール実行）、Channels（Telegram/Discord連携）、skills検索、async hooks、HTTP hooksを追加。価格改定により移行議論が活発化。
- 参照: [https://qiita.com/kotaro_ai_lab/items/4ead93c23597722eef62](https://qiita.com/kotaro_ai_lab/items/4ead93c23597722eef62)

### DeepSeek V4 Pro/Flash (DeepSeek)
- 2026年に超低価格（Pro $0.18、Flash $0.06 per 1M tokens）で高性能を実現。Max/High推論モード搭載。初の外部資金調達で最大73億ドル、評価額515億ドルを目指す。
- 参照: [https://36kr.com/p/3807663643828229](https://36kr.com/p/3807663643828229)

### Gemini Intelligence (Google)

- 参照: [https://www.itmedia.co.jp/news/articles/2605/13/news136.html](https://www.itmedia.co.jp/news/articles/2605/13/news136.html)

- 2026年5月5日リリース。世界初の完全サブクアドラティックLLM。SSAによりO(n²)→O(n)に計算量削減。12Mコンテキスト処理、FlashAttentionより52倍高速化・63%計算量削減を達成。
