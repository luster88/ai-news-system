# 最新AIモデルまとめ - 2026-05-31

## 概要
- 2026年5月時点で、Claude Opus 4.8とGPT-5.5が最高性能モデルとして競合。Anthropicは5月29日にOpus 4.8を一般提供開始し、コーディング・エージェント作業で大幅な性能向上を実現。
- Googleは5月20日のI/O 2026でGemini 3.5シリーズを発表。軽量モデルGemini 3.5 Flashは一部ベンチマークでGemini 3.1 Proを上回り、コーディング・エージェント領域に特化。高性能モデルGemini 3.5 Proは6月公開予定。
- OpenAIはCodex（AIコーディングエージェント）の企業展開を加速。Cisco、Ramp、Braintrustなどが導入し、開発サイクルを数週間に短縮。5月にはAstral買収を発表しPythonエコシステムへの投資を強化。
- コーディング支援AI市場が急拡大。Claude Code、Codex、Cursor Composer 2.5などが競合し、エージェント型開発ツールが主流に。Microsoftはコスト理由で社内Claude Codeライセンスを取り消しGitHub Copilot CLIへの移行を指示。
- セキュリティ特化モデルも登場。OpenAIはGPT-5.5-Cyberを日本政府・金融機関向けに提供開始。AnthropicのClaude Mythos Previewは1ヶ月で10,000件超の脆弱性を発見したが、修正は97件のみと課題も浮上。

## 今週の新モデル・アップデート

### Claude Opus 4.8 (Anthropic)
- 2026年5月29日に一般提供開始。コーディング・エージェント作業で大幅な性能向上。不確実性を明示する「正直さ」が向上し、コード欠陥見逃し率が約4分の1に減少。努力量調整機能とdynamic workflows機能を研究プレビューとして導入。
- 参照: [https://www.anthropic.com/news/claude-opus-4-8](https://www.anthropic.com/news/claude-opus-4-8)

### Gemini 3.5 Flash / Gemini 3.5 Pro (Google)
- 2026年5月20日のGoogle I/Oで発表。Gemini 3.5 Flashは即日提供開始され、コーディング・エージェント領域に特化。一部ベンチマークでGemini 3.1 Proを上回る。出力速度は他社フロンティアモデルの4倍。Gemini 3.5 Proは6月公開予定。
- 参照: [https://www.itmedia.co.jp/aiplus/article/2605/20/2000000010.html](https://www.itmedia.co.jp/aiplus/article/2605/20/2000000010.html)

### Gemini Omni (Google)
- 2026年5月19日発表。テキスト・画像・動画・音声から動画を生成できる新マルチモーダルモデル。Gemini Omni FlashがGeminiアプリ・Google Flow・YouTube Shortsに展開。自然言語での対話を通じて動画編集が可能。
- 参照: [https://deepmind.google/blog/introducing-gemini-omni](https://deepmind.google/blog/introducing-gemini-omni)

### GPT-5.5-Cyber (OpenAI)
- 2026年5月21日、日本政府と一部企業への提供を発表。サイバーセキュリティに特化したモデルで、脆弱性の特定や対応の迅速化を支援。Trusted Access for Cyber (TAC)の枠組みで承認された組織に限定提供。
- 参照: [https://www.itmedia.co.jp/aiplus/article/2605/21/2000000014.html](https://www.itmedia.co.jp/aiplus/article/2605/21/2000000014.html)

### Cursor Composer 2.5 (Cursor)
- 2026年5月18日発表。Kimi K2.5をベースに独自post-trainingを施し、Claude Opus 4.7とほぼ同等のベンチマーク性能（SWE-bench 79.8%）を約1/10のコスト（$0.07/タスク）で実現。現時点ではCursor IDE・CLIのみで利用可能。
- 参照: [https://qiita.com/kai_kou/items/b7adde1f35dfdf942382](https://qiita.com/kai_kou/items/b7adde1f35dfdf942382)

### Claude Code Routines & Channels (Anthropic)
- 2026年4月14日にRoutines（クラウドスケジュールタスク）をリサーチプレビューとして公開。PCを閉じていてもAnthropicのインフラ上でエージェントが動作。GitHubイベントトリガーにも対応。Channels機能でTelegram・Discord・iMessage連携も可能に。
- 参照: [https://qiita.com/goki602/items/48cd1f30af1fb2f3c3c9](https://qiita.com/goki602/items/48cd1f30af1fb2f3c3c9)

### Claude Code Projects & Dispatch (Anthropic)
- 2026年3月20日追加。Projectsはプロジェクトごとにファイル・指示・作業履歴をローカル保存し、コンテキストを継続可能に。Dispatchは外出先からデスクトップのAIエージェントを操作できる機能。MCPコネクター38以上と組み合わせ可能。
- 参照: [https://qiita.com/AI-SKILL-LAB/items/a39ca150b3a784a967fd](https://qiita.com/AI-SKILL-LAB/items/a39ca150b3a784a967fd)

### Google Antigravity 2.0 (Google)
- 2026年5月20日のGoogle I/Oで発表。マルチエージェントオーケストレーションプラットフォームとして一般提供開始。Gemini 3.5に加えてClaude Opus/Sonnet 4.6やGPT-OSS-120Bなど第三者モデルも選択可能な開放的設計。/goal、/grill-me、/scheduleなど新コマンド追加。
- 参照: [https://36kr.com/p/3817576331747331](https://36kr.com/p/3817576331747331)

### OpenAI Astral買収 (OpenAI)
- 2026年3月22日発表。Pythonツールチェーン開発企業Astralを買収。RuffやuvなどのツールでCodexの成長を加速させ、次世代Python開発者ツールを強化する狙い。
- 参照: [https://openai.com/index/openai-to-acquire-astral](https://openai.com/index/openai-to-acquire-astral)

### GitHub Copilot AIクレジット制移行 (Microsoft/OpenAI)
- 2026年6月1日から、プレミアムリクエスト制（回数ベース）からAIクレジット制（トークンベース）の重量課金に移行。個人プランでは月1,500クレジット+フレックス分が含まれ、超過分は$0.01/クレジットで購入可能。
- 参照: [https://qiita.com/shahin0809/items/9c7ec8c7661272c63001](https://qiita.com/shahin0809/items/9c7ec8c7661272c63001)

### Nano Banana 2 / Nano Banana Pro (Google)
- 2026年5月29日に一般提供開始。Nano Banana 2（Gemini 3.1 Flash Image）は動画入力から画像生成可能。Nano Banana Pro（Gemini 3 Pro Image）も同時リリース。Gemini Enterprise Agent Platform経由で提供。
- 参照: [https://www.itmedia.co.jp/news/articles/2605/29/news146.html](https://www.itmedia.co.jp/news/articles/2605/29/news146.html)

### SubQ 1M-Preview (Subquadratic)
- 2026年5月5日リリース。世界初の完全サブクアドラティックアーキテクチャ採用LLM。SSA（Subquadratic Sparse Attention）により計算量をO(n)に削減。12Mトークンのコンテキスト処理を実現し、FlashAttentionより52倍高速化。
- 参照: [https://qiita.com/kai_kou/items/9428b9cc23475970c518](https://qiita.com/kai_kou/items/9428b9cc23475970c518)
