## 今週の新モデル・アップデート

### Claude Opus 4.8 (Anthropic)
- 2026年5月28日リリース。「正直さ」を大幅強化し、コードの欠陥見逃し率を前バージョンの約1/4に削減。動的ワークフロー機能追加で大規模問題に対応。高速モードは2.5倍速・コスト1/3を実現。
- 参照: [https://www.anthropic.com/news/claude-opus-4-8](https://www.anthropic.com/news/claude-opus-4-8)

### Cursor Composer 2.5 (Cursor)
- 2026年5月18日発表。Kimi K2.5ベースに独自post-training。Claude Opus 4.7と同等性能を約1/10コスト（$0.07 vs $4.10/タスク）で実現。SWE-bench Multilingualで79.8%達成。

### Nano Banana 2 / Pro (Google)
- 2026年5月29日一般提供開始。Nano Banana 2は動画入力から画像生成可能に。Gemini Enterprise Agent Platform経由で提供され、エンタープライズSLA適用。
- 参照: [https://www.itmedia.co.jp/news/articles/2605/29/news146.html](https://www.itmedia.co.jp/news/articles/2605/29/news146.html)

### GPT-5.4 mini/nano (OpenAI)
- 2026年3月22日発表。コーディング、ツール利用、マルチモーダル推論、大量API呼び出しに最適化された小型高速モデル。詳細な性能指標は未公開。
- 参照: [https://openai.com/index/introducing-gpt-5-4-mini-and-nano](https://openai.com/index/introducing-gpt-5-4-mini-and-nano)

### Claude Sonnet 4.6 (Anthropic)
- 2026年2月17日発表。従来Opusクラスが必要だったタスクをSonnetで処理可能に。100万トークンコンテキストをベータ提供。価格は従来のSonnet 4.5と同じを維持。
- 参照: [https://www.anthropic.com/news/claude-sonnet-4-6](https://www.anthropic.com/news/claude-sonnet-4-6)

### Project Glasswing拡大 (Anthropic)
- 2026年6月2日発表。Claude Mythos Previewの提供対象を約50組織から約150組織に拡大。1ヶ月で10,000件以上の高・重大度脆弱性を発見。日本政府もアクセス権取得。
- 参照: [https://www.anthropic.com/news/expanding-project-glasswing](https://www.anthropic.com/news/expanding-project-glasswing)

### Codex機能拡張 (OpenAI)
- 2026年6月2日発表。開発者以外の職種向けに6つの職種別プラグイン導入。62アプリ・110スキル統合。週間利用者500万人のうち約20%が非開発者で、開発者の3倍以上の成長率。
- 参照: [https://openai.com/index/codex-for-every-role-tool-workflow](https://openai.com/index/codex-for-every-role-tool-workflow)

### GPT-Rosalind新機能 (OpenAI)
- 2026年6月4日発表。GPT-5.5のエージェント機能に創薬化学・ゲノミクス専門知識を統合。LifeSciBenchベンチマークで6つのワークフロー領域で性能向上確認。研究プレビュー版提供中。
- 参照: [https://openai.com/index/introducing-new-capabilities-to-gpt-rosalind](https://openai.com/index/introducing-new-capabilities-to-gpt-rosalind)

### Gemini 3.5 Flash (Google)
- 2026年発表。速度と性能のバランスに優れたモデル。検索・グラウンディング機能強化。Batch APIで50%コスト削減。入力$1.50/出力$9.00（標準）、Batchは半額。
- 参照: [https://ai.google.dev/pricing](https://ai.google.dev/pricing)

### Claude Code /code-review --fix (Anthropic)
- 2026年5月27日、v2.1.152で追加。レビュー後に修正を作業ツリーへ自動適用し、再利用・簡略化・効率化の提案まで実行。/simplifyコマンドはエイリアスに。

### Claude Cowork Projects + Dispatch (Anthropic)
- 2026年3月20日追加。Projectsはプロジェクトごとにファイル・指示・履歴をローカル保存。Dispatchは外出先からデスクトップAIエージェントを操作可能に。38以上のMCPコネクター対応。

### OpenAI、Astral買収 (OpenAI)
- 2026年3月22日発表。Pythonツールチェーン開発企業Astral（Ruff、uvなど）を買収。Codexの成長加速と次世代Python開発者ツール強化が狙い。

### 日本サイバー・アクションプラン (OpenAI)
