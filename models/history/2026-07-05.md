# 最新AIモデルまとめ - 2026-07-05

## 概要
- 2026年7月時点で、Claude Fable 5とClaude Opus 4.8が最高性能モデルとして君臨。Fable 5は米政府規制により一時停止後、7月1日に全世界で再展開された。
- OpenAIはGPT-5.6シリーズ（Sol/Terra/Luna）を発表し、政府要請により段階的公開を実施。Solはコーディング・サイバーセキュリティで高性能を発揮。
- 中国勢ではGLM-5.2が高評価を獲得し、DeepSeek V4 ProがDSpark技術で推論速度60-85%向上。Kimi K2.7も先進モデルとして国際的に認知された。
- コーディングエージェント市場が急拡大。Claude Code、OpenAI Codex、Cursorなどが企業導入を加速し、AIが「デジタル同僚」として定着しつつある。
- 価格競争が激化し、二極化が進行。高性能モデル（Fable 5等）は価格倍増でも需要があり、低価格モデル（DeepSeek等）がコスト削減ニーズを獲得。中間層モデルは淘汰の危機。

## 今週の新モデル・アップデート

### Claude Fable 5 (Anthropic)
- 6月12日に米政府の輸出規制で一時停止後、7月1日に全世界で再展開。ジェイルブレイク対策を強化し、Amazon・Microsoft・Googleと共同で業界標準フレームワークを提案。7月7日まで週間上限の50%まで利用可能。
- 参照: [https://www.anthropic.com/news/redeploying-fable-5](https://www.anthropic.com/news/redeploying-fable-5)

### Claude Sonnet 5 (Anthropic)
- 6月30日発表。Opus 4.8に近い性能で価格は低く、前モデルSonnet 4.6から推論・ツール使用・コーディングで大幅改善。API価格は8月末まで入力$2/百万トークン、出力$10/百万トークン（その後$3/$15）。
- 参照: [https://www.anthropic.com/news/claude-sonnet-5](https://www.anthropic.com/news/claude-sonnet-5)

### Claude Science (Anthropic)
- 6月30日発表。科学研究者向けAIワークベンチ。PubMed・Jupyter・Rなど断片化ツールを統合し、文献分析から論文作成まで一貫実行。60以上のスキルとコネクタ搭載。macOS・Linux・SSH・HPCで利用可能。
- 参照: [https://www.anthropic.com/news/claude-science-ai-workbench](https://www.anthropic.com/news/claude-science-ai-workbench)

### GPT-5.6 (Sol/Terra/Luna) (OpenAI)
- 6月29日発表。米政府要請により限定プレビューで段階公開。Solはコーディング・サイバーセキュリティ・生物学特化。Terraは5.5同等性能で2倍安価、Lunaは最低コスト。数週間後に一般提供予定。
- 参照: [https://openai.com/index/previewing-gpt-5-6-sol](https://openai.com/index/previewing-gpt-5-6-sol)

### DeepSeek V4 Pro DSpark (DeepSeek)
- 6月27日発表。推測的デコーディング技術DSpark導入で推論速度60-85%向上。軽量草稿モデルで候補トークン並列生成し目標モデルで一括検証。技術論文とオープンソースフレームワーク「DeepSpec」同時公開。
- 参照: [https://36kr.com/p/3871187114448133](https://36kr.com/p/3871187114448133)

### Gemini 3.5 Flash (コンピュータ操作) (Google)
- 6月24日発表。以前は独立したGemini 2.5でのみ提供されていたコンピュータ操作機能をメインのFlashモデルに統合。ブラウザ・モバイル・デスクトップ環境横断で画面認識・推論・操作が可能。
- 参照: [https://deepmind.google/blog/introducing-computer-use-in-gemini-3-5-flash](https://deepmind.google/blog/introducing-computer-use-in-gemini-3-5-flash)

### Nano Banana 2 Lite / Gemini Omni Flash (Google)
- 6月30日発表。Nano Banana 2 Liteは最速・最コスト効率の画像生成モデル。Gemini Omni Flashは高品質動画生成と対話的編集が可能。両モデルともGoogle AI Studio・Gemini API・Gemini Enterprise Agent Platformで利用可能。
- 参照: [https://deepmind.google/blog/start-building-with-nano-banana-2-lite-and-gemini-omni-flash](https://deepmind.google/blog/start-building-with-nano-banana-2-lite-and-gemini-omni-flash)

### Sarashina3 (SB Intuitions (ソフトバンク))
- 6月30日提供開始。30兆トークン超で事前学習した国産LLM。標準モデル「mini」、軽量「nano」、ガードモデル、埋め込みモデル、リランキングモデルの計5種類。Oracle基盤の国内データセンターで運用。
- 参照: [https://www.itmedia.co.jp/aiplus/article/2606/30/2000000143](https://www.itmedia.co.jp/aiplus/article/2606/30/2000000143)

### Interactions API (GA) (Google)
- 6月22日正式リリース。Geminiモデルとエージェント向けデフォルトインターフェース。従来のgenerateContent APIに代わり、ロールベース構造から型付きステップ方式に簡素化。今後の新機能はこのAPIのみで提供。
- 参照: [https://the-decoder.com/google-makes-interactions-api-the-default-interface-for-gemini-models-and-agents](https://the-decoder.com/google-makes-interactions-api-the-default-interface-for-gemini-models-and-agents)

### Claude Code システムプロンプト80%削減 (Anthropic)
- 7月2日発表。新しいFable 5モデル（Mythosクラス）はより少ない指示と例で動作し、詳細なガイドラインが創造性を制約するため、システムプロンプトを80%削減。厳格なルールではなくコンテキストで誘導する方針に転換。
- 参照: [https://the-decoder.com/anthropic-says-it-cut-80-percent-of-claude-codes-system-prompt-because-fable-5-models-want-a-smaller-system-prompt](https://the-decoder.com/anthropic-says-it-cut-80-percent-of-claude-codes-system-prompt-because-fable-5-models-want-a-smaller-system-prompt)
